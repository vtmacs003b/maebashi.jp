プログラミング体験について

プログラミングを子どもにやらせてみたいけど、何をやらせていいか分からない。 プログラミングに興味があるけど、親がなかなかパソコンを使わせてくれない。 そういった小中学生に少しでもいいから、プログラミングに触れてもらいたい、そのために夏休みに一日だけプログラミングを体験できる機会を用意しました。

機材はマイクロソフトより全部貸与されたものなので、パソコン等は一切必要なし。参加費も頂きません。当日は手ぶらできて頂いて構いません。
実際にマイクロソフトで開発者として活躍されたこともある鵜飼佑氏をお呼びして、プログラミングについてお話しして頂く時間もあります。

Hour of Code について

Hour of Code は、「コード（プログラミング）」の謎を解き明かし、全ての人が基本を学ぶことができることを示し、コンピューターサイエンス分野の幅広い参加者に向けて作成された、1時間のコンピューターサイエンス入門として始まりました（Hour of Code のサイトより引用）。
Minecraft Hour of Code は Minecraft の世界を楽しみながら、プログラミングを体験できます。
アドベンチャーとデザイナーの２種類があります。

Code Builder for Minecraft について

Minecraft はブロックを地面や空中に配置し、自由な形の建造物等を作っていくゲームです。
小中学生を中心に世界的に人気なゲームでパソコン以外でも各種ゲーム機で遊べるようになっています。
Code Builder for Minecraft は Minecraft の教育版 Minecraft: Education Edition で動作するプログラミング環境です。今年５月に公開されたばかりなので、まだあまり一般的ではありませんが、ゲームの世界でプログラミングを体験できるので、楽しくプログラミングを学ぶことができます。
なお、体験では Minecraft の詳細な操作等の説明は行いませんので、ご了承下さい。

Q:プログラミングは初めてですけど、大丈夫ですか？
A:今回はプログラミングを体験してもらう機会としたいので、初心者は大歓迎です。
Q:プログラミングはともかくパソコンを全く触ったことがありません、大丈夫ですか？
A:小学生対象にはなりますが、Minecraft Hour of Code はパソコンに初めて触る子でもすぐに慣れます。
Q:マインクラフトはやったことがありませんが、大丈夫ですか？
A:小学生対象にはなりますが、Minecraft Hour of Code はならば問題ありません。Code Builder はパソコン版のマインクラフトの操作方法がある程度分かることを前提として話を進めていきます。
Q:パソコンを持ってくる必要はありますか？
A:パソコンはすべて日本マイクロソフトから貸与されたものを使います。手ぶらでお越し頂いて構いません。個人でお持ちしたい場合もあるかと思いますが、Code Builder の環境がインストールされていないと思いますので、推奨できません。
Q:インターネットは繋がりますか？
A:インターネット環境は日本マイクロソフトの機材を用いるので、日本マイクロソフトから貸与されているパソコンのみ使うことができます。仮に個人でお持ちしたパソコンを使いたい場合、各自でインターネット環境を用意して下さい。
Q:開発者からのお話とは何ですか？
A:Minecraft Hour of Code と Code Builder for Minecraft のどちらにも開発で関わっている鵜飼佑氏から開発時に苦労した点やプログラミングの面白さについて話をして頂きます。質疑応答の時間も作る予定です。
Q:会場で飲食は可能ですか？
A:お借りしているセミナー室は飲食禁止となっております。当日は近くにパソコンがある、ということもあるので、飲食される場合はセミナー室の外でお願い致します。
Q:対象外の時間に参加することは可能ですか？
A:時間毎に対象が小学生と中学生に分かれていますので、それに合わせて下さい。ただ、対象外の方が周りから見ているということは可能です。
Q:保護者は同伴した方が良いのでしょうか？
A:小学校低学年までは保護者同伴でお願い致します。小学校高学年、中学生はお子様一人で参加できます（保護者の同伴も可能です）。
Q:会場まで車で行くつもりですが、駐車場はありますか？
A:セキスイハイムビルは地下駐車場が広いので、おそらく問題ないかと思います。
