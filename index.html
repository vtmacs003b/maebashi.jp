すべての子どもにプログラミングを

CoderDojo前橋（CoderDojo Maebashi）は、群馬県前橋市で開催を予定している小学生・中学生向けプログラミングクラブです。
前橋市の会場で、２０１７年４月より毎月１回の定期開催を予定しています。

CoderDojo とは？

CoderDojo は7〜17歳の子どもを対象にしたプログラミング道場です。2011年にアイルランドで始まり、世界では70カ国・1,200の道場、日本では全国に70以上の道場があります (2017年1月)。
CoderDojo 前橋は、群馬県初かつ唯一（２０１７年現在）の CoderDojo で、主に Raspberry Pi を用いてプログラミングをする予定です。

CoderDojo 前橋について

CoderDojo 前橋は、２０１７年４月より活動開始予定です。
内容は、小型コンピュータ Raspberry Pi を用いたプログラミングが中心となります。
Raspberry Pi - Wikipedia
Raspberry Pi ｜ラズパイ正規総代理店 RSコンポーネンツ
CoderDojo はボランティア活動になりますので、参加費は頂きません。無償で参加することが可能です（ただし、Raspberry Pi などの必要機材は購入して頂くことになります）。
参加募集定員は２０名の予定で、場所は前橋市立永明公民館になります。

メンター募集中

現在、CoderDojo 前橋では、一緒に活動してくれるメンターを募集しています。 Raspberry Pi を持っていなくても、Scratch を使ったことがある方なら大歓迎です。 下記メールアドレスまで連絡をお願いいたします。

maebashi.jp@coderdojo.com

参加者募集について

CoderDojo 前橋の参加者募集については、２月上旬をめどにこちらでご案内いたします。
現状では
参加費：無料（ただし、Raspberry Piなど機材費で初期費用が5,000円〜15,000円程度かかります）
場所：前橋市立永明公民館
定員：２０名
２０１７年４月２日（日）に保護者対象の説明会を実施予定です。
