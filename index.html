すべての子どもにプログラミングを

CoderDojo（コーダードージョー） 前橋は、参加費無料でプログラミングを学べる小中学生 を対象としたコンピュータサイエンス道場です。プログラミングを始めとしたコンピュータサイエンスは子どもたちがこれからの社会で生きていくために必要な能力です。
２０２０年からは小学校でプログラミングが必修化されます。

CoderDojo とは？

CoderDojo は7〜17歳の子どもを対象にしたプログラミング道場です。2011年にアイルランドで始まり、世界では70カ国・1,200の道場、日本では全国に70以上の道場があります (2017年1月)。
CoderDojo 前橋は、群馬県初かつ唯一（２０１７年現在）の CoderDojo で、主に Raspberry Pi を用いてプログラミングをする予定です。
詳しい道場の雰囲気については、次のスライドを参考にして下さい。

CoderDojo 前橋について

CoderDojo 前橋は、２０１７年４月より活動開始予定です。
内容は、小型コンピュータ Raspberry Pi を用いたプログラミングが中心となります。
Raspberry Pi - Wikipedia
Raspberry Pi ｜ラズパイ正規総代理店 RSコンポーネンツ
CoderDojo はボランティア活動になりますので、参加費は頂きません。
ただし、施設利用料として年間500円頂きます。
また、別途実費として Raspberry Pi の購入等が必要になります。
参加募集定員は２０名の予定です。

CoderDojo 代表：関口　悠
群馬県前橋市在住。
県内の高校で数学教師をしている、一児の父。
マイクロソフト認定の教育イノベーターとしても活動している。

メンター募集中

現在、CoderDojo 前橋では、一緒に活動してくれるメンターを募集しています。 Raspberry Pi を持っていなくても、Scratch や Minecraft を使ったことがある方なら大歓迎です。

メンターとして登録して頂いた方にはもれなく Raspberry Pi を進呈いたします。 メンター希望の方は、下記まで連絡をお願いいたします。

maebashi.jp@coderdojo.com
