すべての子どもにプログラミングを

CoderDojo（コーダードージョー） 前橋は、参加費無料でプログラミングを学べる小中学生を対象としたコンピュータサイエンス道場です。情報化が進む現代、プログラミングを始めとしたコンピュータサイエンスは子どもたちが社会で生きていくために必要な能力です。
参考：情報化と教育（文部科学省）、コンピュータサイエンスってなんだろう（かながわグローバルIT研究所）

２０２０年からは小学校でプログラミングが必修化され、２０２５年１月には「大学入学共通テスト」に基礎的な科目としてプログラミングなどの情報科目が追加される予定です。
参考：大学新入試にプログラミング　首相「現代のそろばん」（日本経済新聞）

CoderDojo とは？

CoderDojo は7〜17歳の子どもを対象にしたプログラミング道場です。2011年にアイルランドで始まり、世界では86カ国・1,400の道場、日本では全国に140以上の道場があります (2018年5月現在)。
CoderDojo 前橋は、群馬県初の CoderDojo です。

CoderDojo 前橋について

CoderDojo 前橋は、２０１７年４月より月１回活動しています。
CoderDojo 前橋はボランティア活動になりますので、参加費は頂きませんが、参加にはプログラミング環境がインストールされたパソコン等の持ち込みが必要となります（機材の貸し出しもいたします）。
CoderDojo 前橋では比較的安価に購入でき、またプログラミング環境のインストールが不要な Raspberry Pi を推奨していますが、既にプログラミング環境がインストールされたパソコンや iPad などのタブレット端末をお持ちの場合はそちらを使用することも可能です。
Raspberry Pi - Wikipedia
Raspberry Pi ｜ラズパイ正規総代理店 RSコンポーネンツ
なお、会場にはインターネット環境がありませんので、CoderDojo 前橋の活動中にプログラミング環境のインストールはできませんのでご注意ください。
また、参加したときに Raspberry Pi を貸し出すことも可能ですが、台数に限りがございますので、各自で購入・持参していただけると助かります。

メンター募集について

現在、CoderDojo 前橋では、一緒に活動してくれるメンターを募集しています。 Raspberry Pi を持っていなくても、Scratch や Minecraft を使ったことがある方なら大歓迎です。

メンターとして登録して頂いた方にはもれなく Raspberry Pi を進呈いたします。 メンター希望の方は、下記メールアドレスまで連絡をお願いいたします。
